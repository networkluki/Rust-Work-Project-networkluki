## Repository layout

Current structure:

- `oxide/plugins/` — Oxide/uMod C# plugins used by the server.
- `Server-Info` — server information/config-related file.
- `frostraid.png` — image asset.
- `dc` — project-specific helper/resource file.

- [`docs/REPO_STRUCTURE.md`](docs/REPO_STRUCTURE.md)
